# PostGres
Романов Артем Ігорович 
Програма запускає базу даних PostgreSQL у Docker та виконує основні операції з таблицею:

- додавання одного та кількох записів;
- оновлення одного та кількох записів;
- видалення одного та кількох записів;
- вибірка даних за умовою;
- підрахунок кількості записів.

База створюється автоматично через EF Core.  
Для запуску:
#docker compose up -d
#dotnet run
